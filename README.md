### ソースコードの変更点が反映されない場合

端末を振るとExpoのメニューが出て来るので、以下のどれかを試してみましょう

1. Refresh
2. Go To Expo Home して QRコードを再読込
3. メニューが出ない場合は、アプリのメモリを開放して再起動
